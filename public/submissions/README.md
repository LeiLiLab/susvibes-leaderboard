# SusVibes Leaderboard Submissions

This directory contains model evaluation submissions for the SusVibes leaderboard.

## Directory Structure

Each submission is stored in its own directory following the naming convention:
`{model-name}_{agent-framework}_{submission-type}_{date}/`

## How to Submit

For detailed submission instructions, please refer to the [Submission Guide](../../SUBMISSION_GUIDE.md) in the root directory.

## Questions?

If you have questions about submitting results, please:
1. Check the [SusVibes documentation](https://github.com/LeiLiLab/susvibes)
2. Open an issue in this repository
